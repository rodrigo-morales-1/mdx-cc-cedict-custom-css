The file =main.py= contains Python code to create a new =*.mdx= dictionary from the dictionary =cc-cedict_enlarged_characters.mdx= which can be downloaded at [[https://github.com/lxs602/Chinese-Mandarin-Dictionaries/tree/main/CC-CEDICT%20(with%20enlarged%20characters)][lxs602/Chinese-Mandarin-Dictionaries]] . * Instructions for running the Python script

1. Download the file =cc-cedict_enlarged_characters.mdx= from [[https://github.com/lxs602/Chinese-Mandarin-Dictionaries/tree/main/CC-CEDICT%20(with%20enlarged%20characters)][lxs602/Chinese-Mandarin-Dictionaries]]

1. Run the Python script to create the file =mdx-entries.txt=

2. Run the command below to create =cc-cedict-custom-css.mdx=

#+BEGIN_SRC sh
mdict --title mdx-title.txt --description mdx-description.txt -a mdx-entries.txt cc-cedict-custom-css.mdx
#+END_SRC
